Module Tester

module Tester: sig .. end

Tester is a unit test framework for OCaml. It allows the easy creation of unit tests for OCaml code. It is somewhat based on OUnit2, another unit testing framework for OCaml. However, its output can be customized per test case, and it is easier to see whether a case has passed or failed.

type 'a test = {
   compare_fun : 'a -> 'a -> bool;
   string_of_result : 'a -> string;
   input : string;
   expected_result : 'a;
   actual_result : 'a;
}

The record type that is associated with a test; it consists of compare_fun a function to compare the expected and actual results of the function, string_of_result a function to parse the output of the function, input the string representing the input to the function, expected_result the expected output of the function, and actual_result the value that the function actually returned
